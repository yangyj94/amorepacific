css 작성 가이드

  • css 작성시 레이아웃 속성 - 형태속성 - 위치속성 - 사이즈 및 컬러 - 기타 순으로 통일성있게 작성한다.
  • 또한, 스타일을 구성하는 속성끼리 묶어서 작성한다.
  • 컴포넌트 일경우 부모 분기를 태우지 않고 작성한다.(다른곳에서도 사용에 용이하도록 작성)
  • 영역별 스타일을 쉽게 확인 할수 있도록 주석을 활용한다.
  • 용량 최소화를 위해 속성간 공백은 제거하여 작성한다.
