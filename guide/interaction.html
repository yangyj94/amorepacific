인터렉션 실행 가이드

1. header nav

  • div.wrap-menu에 open클래스 부여시 레이어 활성화 됩니다.
  • 클릭시에도 활성화 될 수 있게 스크립트 적용시켰습니다.

2. scroll

  • window scrollTop > 1 = top버튼 노출 / 그 이하일시 top버튼 숨김처리했습니다.
  • top버튼 클릭시 맨위로 이동됩니다.
  • window scrollTop > div.store-info offset top = fixed 헤더 노출, 기존헤더는 aria-hidden 처리했습니다.
  • 탭 영역은 position:sticky로 스타일 처리했습니다.

3. swiper

  • 비쥬얼 영역, 홈 > 매장소식 영역 swiper로 작업하였습니다.
  • swiper영역 hover시 이전,다음 페이지 이동 버튼 노출됩니다.
  • 전화, 예약, 공유 버튼 영역은 현재 화면으로 고려하였을때 최소사이즈까지 정상노출되므로 swiper추가하지 않았습니다.
